# 🩺 Data Doctor Pro

A simple and interactive **CSV Data Analysis & Cleaning Tool** built using **Python, Pandas, NumPy, and Flask**. Upload any CSV file to analyze its quality, detect missing values and duplicates, clean the dataset, and download the cleaned version.

---

## 📌 Features

- 📂 Upload CSV files
- 📊 Dataset overview
  - Total Rows
  - Total Columns
  - Memory Usage
- 💯 Data Health Score (0–100)
- 📋 Dataset Preview
- 🏷️ Column Names Display
- ⚠️ Missing Value Analysis
- 📈 Summary Statistics
  - Average
  - Minimum
  - Maximum
- 🧹 Automatic Data Cleaning
  - Removes duplicate rows
  - Fills missing numeric values with mean
  - Fills missing text values with "Unknown"
- ⬇️ Download Cleaned CSV

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/DataDoctorPro.git
```

### Open the project

```bash
cd DataDoctorPro
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
